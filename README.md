# 🕒 Crizmodo – The Ultimate Discord Timer Bot

**Crizmodo** is a powerful, community-tested Pomodoro timer bot for Discord, designed to boost productivity with structured work and break sessions. Built with collaborative features and a clean, interactive interface, Crizmodo keeps your server focused and motivated.

> 🚀 **Currently used in a server with over 50,000+ members**

---

## ✨ Features

- ⏱️ **Pomodoro Timer** — Start focused study sessions with built-in short/long breaks.
- 👥 **Group Sessions** — Host synced Pomodoro cycles for teams or communities.
- 💬 **Dynamic Embeds** — Beautifully updated messages with progress bars, timers, and status.
- 🔘 **Interactive Controls** — Use buttons to skip phases or stop sessions anytime.
- 🔄 **Single-Message Dashboard** — All session updates happen in the same embed (no spam).
- 📢 **Live Reminders** — Periodic pings to keep everyone on track.
- 🛡️ **Reliable & Lightweight** — Designed for performance in large-scale communities.

---

## ⚙️ Commands Overview

| Command             | Description                                      |
|---------------------|--------------------------------------------------|
| `/pomodoro start`   | Start a solo Pomodoro session                    |
| `/pomodoro study`   | Shortcut to begin a focused work phase          |
| `/pomodoro break`   | Take a short break manually                      |
| `/pomodoro group`   | Start or join a group Pomodoro session          |
| `/pomodoro stop`    | Ends the current session                        |
| `/pomodoro help`    | View usage instructions                         |

---

## 📸 Sample UI

> Crizmodo uses **Discord Embeds** with:
- Current session phase
- Countdown timer with live Discord timestamps
- Visual progress bar (`█ █ ░ ░` style)
- Control buttons: **Skip**, **Stop**, etc.

## 🚀 Deployment

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/crizmodo.git
   cd crizmodo
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Create `.env` file:

   ```
   TOKEN=your_discord_bot_token
   CLIENT_ID=your_bot_client_id
   ```

4. Run the bot:

   ```bash
   node index.js
   ```
